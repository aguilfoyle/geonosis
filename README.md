# Geonosis

> "The droid factories of Geonosis will build your army."

AI-powered software development orchestration system.

## Prerequisites

- Node.js 18+
- Python 3.11+
- Docker & Docker Compose
- pnpm (recommended)

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development environment
pnpm dev
```

## Architecture

This is a monorepo managed by Turborepo with the following structure:

- **apps/api** - Python FastAPI backend
- **apps/ui** - Next.js frontend
- **docs/agents** - Agent rule documentation
- **packages/** - Shared packages and utilities
